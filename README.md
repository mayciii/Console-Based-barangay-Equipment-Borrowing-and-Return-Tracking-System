## Barangay Equipment Borrowing System
A console-based Java OOP system for tracking equipment borrowed by barangay residents.

### Features
- Role-based login (Admin / Borrower)
- Borrow and return equipment
- Transaction logging with timestamps
- File-based data persistence

### How to Run
javac *.java
java Main

### Classes
- `User` — base class for all users
- `Admin` — manages equipment and views logs
- `Borrower` — borrows and returns equipment
- `Equipment` — tracks item availability
